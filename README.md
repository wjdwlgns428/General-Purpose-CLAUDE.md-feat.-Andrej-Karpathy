# General-purpose CLAUDE.md

A general-purpose adaptation of the viral CLAUDE.md inspired by Andrej Karpathy’s comments on LLM's pitfalls. It keeps the spirit of careful, goal-aware AI collaboration while making the assistant less tied to coding-specific rules and more capable of appropriate autonomy.

## About

This repository is a general-purpose adaptation of the `CLAUDE.md` idea that went viral after Andrej Karpathy shared his experience using Claude Code.

The original `CLAUDE.md` was primarily designed for software development. It helped Claude behave more carefully, avoid unnecessary changes, and stay focused on concrete engineering goals. This version keeps the core spirit of that approach, but makes it less tied to coding-specific workflows.

I also removed the original second principle, “Simplicity First.” Simplicity is still valuable, but in general-purpose work, the shortest or most minimal answer is not always the best one. Sometimes the assistant should infer context, suggest better alternatives, reframe the problem, or take initiative when doing so clearly helps the user.

The goal is to make AI a more thoughtful collaborator: grounded, transparent about uncertainty, and capable of using an appropriate degree of autonomy.

This file is a starting point. Adapt it to your own workflow.

## What changed from the original?

- Broadened the scope beyond coding-specific workflows
- Removed the original “Simplicity First” principle
- Preserved the emphasis on careful reasoning, uncertainty, and goal-driven execution
- Allows more room for useful autonomy when the context supports it

## How to use

Place the `CLAUDE.md` file in the root of a project or workspace where your AI assistant can read it as part of its working context.

You can use it as-is, or modify the principles to match your own preferred style of AI collaboration.
